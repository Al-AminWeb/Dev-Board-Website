Question-1: What are the different ways to select an element in the DOM?

There are several methods to select HTML elements in the Document Object Model (DOM).

  • getElementById() - Selects an element by its ID
  • getElementsByClassName() - Selects elements by class name.
  • querySelector() - Selects the first matching element using a CSS selector.
  • querySelectorAll() -Selects all matching elements using a CSS selector.
  • getElementsByTagName() - Selects elements by tag name.ред

The above five methods are used more frequently. Besides these, there are some other methods as well.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

There are some differences between innerHTML and innerText and textContent.

  • innerHTML - innerHTML is used for capturing HTML elements. sometimes it is used for styling. Sometimes it is used appending text to the element. it is used but not much.
  • innerText - innerText is used for applying text.We use it for adding text to the element and we can change element text by using this method.
  • textContent - it is very similar to innerText. Using it we can get or set the text content of an HTML element. We can change text by using this method.

Question-3: What is event delegation in the DOM?

DOM Delegation is a technique in JavaScript where instead of adding event listeners to multiple individual elements, you add a single event listener to a parent element. This parent then "delegates" the event to its child elements using event bubbling.
